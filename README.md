# Py_OCR

<p align="center"><img src="assets\logo.png"></p>

> Py_OCR is an OCR tool used to scan documents and pdfs and sort through specified data from the scan.


## Installation



```sh
pip install -r requirements.txt
```
Py_OCR Only works with python3

## Usage example

```sh
python Py_OCR.py 
```
